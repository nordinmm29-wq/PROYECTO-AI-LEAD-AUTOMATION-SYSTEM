# PROYECTO-AI-LEAD-AUTOMATION-SYSTEM
# AI Lead Automation System

Automatización inteligente para la captación, clasificación y seguimiento de leads mediante IA y workflows automatizados.

Este proyecto utiliza n8n para conectar diferentes servicios y crear un flujo de trabajo capaz de recibir leads desde formularios, analizarlos con inteligencia artificial, almacenarlos en un CRM y ejecutar acciones automáticas como envío de emails y seguimiento comercial.

## Funcionalidades
- Captura automática de leads
- Clasificación de leads con IA
- Envío automático de emails
- Seguimiento automatizado
- Gestión centralizada de clientes potenciales

## Stack tecnológico
- n8n
- Google Forms
- Google Sheets
- Gmail
- OpenAI API

## Objetivo
Optimizar procesos comerciales reduciendo tareas manuales y mejorando la eficiencia en la gestión de clientes potenciales.
