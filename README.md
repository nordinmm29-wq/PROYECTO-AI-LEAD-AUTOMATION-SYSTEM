# PROYECTO-AI-LEAD-AUTOMATION-SYSTEM
# AI Lead Automation System

Automatización inteligente para la captación, clasificación y seguimiento de leads mediante IA y workflows automatizados.

Este proyecto utiliza n8n para conectar diferentes servicios y crear un flujo de trabajo capaz de recibir leads desde formularios, analizarlos con inteligencia artificial, almacenarlos en un CRM y ejecutar acciones automáticas como envío de emails y seguimiento comercial.

## Funcionalidades
Descripción del sistema y qué hace exactamente
Diagrama ASCII del flujo completo de nodos
Tabla de todos los nodos agrupados por función (triggers, IA, almacenamiento, decisión, salida)
El prompt exacto que se envía a OpenAI GPT
Tabla de integraciones con el tipo de credencial de cada servicio
Guía de instalación paso a paso incluyendo el comando CLI de n8n
Esquema de columnas de Google Sheets necesario
Tabla de scoring (Alta ≥70 / Media 40-69 / Baja <39) con las acciones que dispara cada nivel
Variables de entorno recomendadas con los IDs reales extraídos del JSON
Sección de personalización para cambiar threshold, campos, períodos de espera y emails
Badges de GitHub al pie con todas las tecnologías usadas

## Stack tecnológico
- n8n
- Google Forms
- Google Sheets
- Gmail
- OpenAI API
- Hubspot
- Slack

## Objetivo
Optimizar procesos comerciales reduciendo tareas manuales y mejorando la eficiencia en la gestión de clientes potenciales.
